# Task ID: 28
# Title: Implementación de la visualización de resultados de búsqueda
# Status: pending
# Dependencies: 26, 27
# Priority: medium
# Description: Desarrollar la interfaz para mostrar resultados de búsqueda según los criterios de la HU007
# Details:
1. Crear componente para mostrar lista de resultados
2. Implementar visualización de datos por documento:
   - Título del documento
   - Número de expediente
   - Fecha de creación
   - Categoría
3. Diseñar sistema de paginación para más de 10 resultados
4. Implementar mensaje para búsquedas sin resultados
5. Crear tooltip o resumen al pasar el mouse sobre un documento
6. Configurar redirección a vista detalle al hacer clic en un documento
7. Implementar ordenamiento de resultados
8. Añadir indicadores visuales para documentos recientes o importantes
9. Optimizar renderizado para listas grandes

# Test Strategy:
Verificar que todos los datos requeridos se muestran correctamente. Probar la paginación con diferentes cantidades de resultados. Comprobar que el tooltip muestra la descripción al pasar el mouse. Verificar la redirección a la vista detalle. Probar el comportamiento cuando no hay resultados.
